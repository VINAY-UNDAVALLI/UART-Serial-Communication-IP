# UART-Serial-Communication-IP
A parameterized UART (Universal Asynchronous Receiver-Transmitter) IP core designed in Verilog HDL for reliable asynchronous serial communication in SoCs and embedded systems. The project implements FSM-based transmitter and receiver modules, configurable baud-rate generation, and oversampling for accurate data reception. A loopback testbench validates correct serial transmission and reception through simulation using Icarus Verilog and GTKWave.

Key Highlights:

FSM-based UART Transmitter (TX) and Receiver (RX),
Configurable baud-rate generator (CLKS_PER_BAUD),
Mid-bit oversampling for robust data sampling,
Loopback verification testbench,
Developed in Verilog HDL and verified with Icarus Verilog & GTKWave.

Learning Outcomes:

UART protocol and asynchronous serial communication,
RTL design in Verilog,
Finite State Machine (FSM) implementation,
Timing analysis and waveform debugging using VCD files.
